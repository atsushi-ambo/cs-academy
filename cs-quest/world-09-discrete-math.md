## 📖 講義 1: 星の言葉 — 論理(15 XP)

**命題**とは真偽が定まる文。論理結合子で組み立てる:

| 記号 | 読み | 意味 |
|---|---|---|
| ¬P | NOT | P でない |
| P ∧ Q | AND | 両方真 |
| P ∨ Q | OR | 少なくとも一方が真 |
| P → Q | ならば | P が真なのに Q が偽、のときだけ偽 |
| P ↔ Q | 同値 | 真偽が一致 |

**P → Q の親戚たち(混同したら塔から落ちる)**:
- **逆** Q → P …… 成り立つとは限らない
- **裏** ¬P → ¬Q …… 成り立つとは限らない
- **対偶** ¬Q → ¬P …… **元と完全に同値**(証明で大活躍)

**述語論理**: ∀x(すべての x について)、∃x(ある x が存在して)。
否定の規則: ¬∀x P(x) ≡ ∃x ¬P(x)。「全員が正しいわけではない」=「間違っている者が少なくとも 1 人いる」。

プログラマにはおなじみだ: `!(a && b) == (!a || !b)`(ド・モルガンの法則)。
論理はコードの条件式・ループ不変条件・仕様記述の文法そのものである。

## 📖 講義 2: 賢者の四剣 — 証明技法(15 XP)

### 🗡️ 直接証明
P を仮定し、定義と既知の定理だけで Q まで歩く。
例: 「偶数 + 偶数 = 偶数」— 2a + 2b = 2(a+b)。終わり。

### 🗡️ 対偶証明
P → Q の代わりに ¬Q → ¬P を示す。
例: 「n² が偶数なら n は偶数」← 対偶「n が奇数なら n² は奇数」は (2k+1)² = 2(2k²+2k)+1 で即座に出る。

### 🗡️ 背理法
結論を否定して矛盾を導く。
**名作: √2 は無理数** — √2 = a/b(既約分数)と仮定すると a² = 2b² より a は偶数、a = 2c とおくと b² = 2c² より b も偶数。既約に矛盾。∎

### 🗡️ 数学的帰納法 — CS 最強の剣
1. **基底**: P(0) を示す
2. **帰納段階**: P(k) を仮定して P(k+1) を示す
3. すると全ての n で P(n) が成り立つ

**強帰納法**は「P(0)〜P(k) すべて」を仮定してよい変種。
再帰アルゴリズムの正しさ・ループ不変条件・データ構造の不変条件の証明は、ほぼすべて帰納法だ。
World 6 の再帰と帰納法は同じものの裏表である(基底ケース=基底、再帰ケース=帰納段階)。

## 📖 講義 3: 星の集まり — 集合・関数・無限(15 XP)

- **集合**: 和 ∪、積 ∩、差 \、補集合、べき集合 P(S)(部分集合全体。|P(S)| = 2^|S|)
- **関係**: 反射律・対称律・推移律をすべて満たす関係が**同値関係**(例: mod 5 で合同)。集合を**同値類に分割**する
- **関数**: 単射(潰れない)・全射(覆い尽くす)・全単射(完全な 1 対 1)
- **濃度**: 全単射が作れる集合同士は「同じ大きさ」。自然数と偶数は同じ大きさ(n ↔ 2n)!

**カントールの対角線論法**: 実数全体は自然数で番号付けできない(非可算)。
0 と 1 の無限列をどう並べても、「i 番目の列の i 桁目を反転」した列は表のどこにもない。
—— この論法は World 13 で**停止問題の決定不能性**として再登場する。伏線を覚えておけ。

## 📖 講義 4: 整数の星座 — 数論入門(15 XP)

- **合同式**: a ≡ b (mod m) ⟺ m が a − b を割り切る。「mod の世界の時計算」
- **最大公約数**: **ユークリッドの互除法** gcd(a, b) = gcd(b, a mod b)。O(log min(a,b)) — 紀元前から現役の最速アルゴリズム
- **拡張互除法**: gcd(a, b) = ax + by となる整数 x, y を求める → **mod 逆元**の計算に使う
- **フェルマーの小定理**: p が素数で gcd(a,p)=1 なら a^(p−1) ≡ 1 (mod p)

これらは飾りではない。**RSA 暗号(大学院篇で攻略)は、この講義の道具だけで組み立てられる。**
ハッシュ関数の設計、乱数生成、誤り訂正符号もすべて数論の上に建つ。

## 📖 講義 5: 数え上げと賭けの星図 — 組合せ論と確率(15 XP)

### 数え上げの四つ道具
- **積の法則**: 独立な選択は掛け算
- **順列** P(n,k) = n!/(n−k)! / **組合せ** C(n,k) = n!/(k!(n−k)!)
- **鳩の巣原理**: n+1 羽の鳩を n 個の巣に入れると、必ず 2 羽入りの巣がある(単純なのに強力。「東京には髪の毛の本数が同じ 2 人が必ずいる」)
- **包除原理**: |A ∪ B| = |A| + |B| − |A ∩ B|

### 離散確率
- **条件付き確率** P(A|B) = P(A∩B)/P(B)、**ベイズの定理** P(A|B) = P(B|A)P(A)/P(B)
- **期待値の線形性**: E[X+Y] = E[X] + E[Y] が**独立でなくても**成り立つ —— 乱択アルゴリズム解析の最終兵器
- **誕生日のパラドックス**: 23 人いれば誕生日被りの確率 > 50%。
  ハッシュ衝突が「思ったよりずっと早く」起きる理由であり、暗号学的ハッシュが 256 ビットも必要な理由(衝突発見は √N = 2^128 回で起こりうる)

### クエスト 1: 四剣の修行(30 XP)

それぞれ指定の剣で証明せよ。

1. (直接)奇数の二乗は奇数である
2. (対偶)整数 n について、3n + 2 が奇数ならば n は奇数である
3. (背理法)素数は無限に存在する(ヒント: 有限個 p₁…pₖ と仮定し、p₁p₂…pₖ + 1 を考える)
4. (帰納法)1 + 2 + ⋯ + n = n(n+1)/2

<details>
<summary>📜 解答の骨子</summary>

1. (2k+1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1 — 奇数の形。
2. 対偶「n が偶数 ⇒ 3n + 2 は偶数」: n = 2k なら 3n + 2 = 2(3k + 1)。
3. 有限個と仮定し N = p₁⋯pₖ + 1 を作ると、N はどの pᵢ でも割り切れない(余り 1)。N の素因数は表にない新顔 — 矛盾。
4. 基底 n=1: 1 = 1·2/2 ✓。帰納: 1+⋯+k+(k+1) = k(k+1)/2 + (k+1) = (k+1)(k+2)/2 ✓。
</details>

### クエスト 4: 賭博師の問答(30 XP)

1. サイコロを 2 個振ったとき、和が 7 になる確率は?
2. ある検査は病気の人を 99% 陽性と判定し、健康な人も 1% 誤って陽性とする。人口の 0.1% が罹患しているとき、陽性判定された人が本当に病気である確率をベイズで求めよ(直感とのズレに驚け)。
3. n 人のパーティで全員がランダムにプレゼントを交換する。自分のプレゼントが自分に戻る人数の期待値を、期待値の線形性で求めよ。

<details>
<summary>📜 解答</summary>

1. 36 通り中 6 通り = **1/6**。
2. P(病|陽) = (0.99 × 0.001) / (0.99 × 0.001 + 0.01 × 0.999) ≈ **9%**。陽性でも 9 割は健康!(偽陽性の海)
3. 各人 i について「自分に戻る」指示確率変数 Xᵢ は E[Xᵢ] = 1/n。線形性より E[ΣXᵢ] = n × 1/n = **1 人**。n によらない!
</details>

## 👹 ボス戦: 塔の頂の「証明竜ロジックドラゴン」(150 XP)

ノートを閉じて挑め。7 割以上で勝利。

1. P → Q の逆・裏・対偶を書き、元の命題と同値なのはどれか述べよ。
2. √3 が無理数であることを背理法で証明せよ。
3. 帰納法で「n ≥ 1 のとき 2ⁿ > n」を証明せよ。
4. カントールの対角線論法の核心(なぜ新しい列が必ず表にないか)を 3 文以内で説明せよ。
5. gcd(252, 105) を互除法で計算過程ごと示せ。
6. 「64 ビットのハッシュなら衝突はまず起きない」という主張を、誕生日のパラドックスで批判せよ。
7. 期待値の線形性が「独立性を仮定しない」ことが、なぜアルゴリズム解析で強力なのか、例を 1 つ挙げて述べよ。

<details>
<summary>📜 解答の骨子</summary>

1. 逆 Q→P、裏 ¬P→¬Q、対偶 ¬Q→¬P。同値なのは**対偶のみ**。
2. √3 = a/b(既約)とすると a² = 3b² → a は 3 の倍数(3 が素数だから)→ a=3c → b² = 3c² → b も 3 の倍数 → 既約に矛盾。
3. 基底: 2¹ = 2 > 1。帰納: 2^(k+1) = 2·2ᵏ > 2k ≥ k+1(k ≥ 1)。
4. 表の i 行 i 桁目を反転した列 D を作ると、D はどの i 行とも i 桁目で食い違う。よってどの行とも一致せず、表は実数(無限列)全体を覆えていない。
5. gcd(252,105) = gcd(105,42) = gcd(42,21) = gcd(21,0) = **21**。
6. 衝突は約 √(2⁶⁴) = 2³² ≈ 43 億回の試行で 5 割の確率で発生する。現代の計算機なら数秒〜数分で到達可能な回数であり「まず起きない」は誤り。
7. 例: クイックソートの比較回数の期待値解析。個々の比較が起きる事象は互いに複雑に依存するが、線形性により「各ペアが比較される確率」を独立に足し合わせるだけで O(n log n) が導ける。
</details>
