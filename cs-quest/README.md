# 🎮 CS Quest 〜コンピュータサイエンス冒険譚〜

**🌐 言語 / Language: 日本語(このページ) ・ [English](https://github.com/jwasham/coding-interview-university)**

> ようこそ、冒険者よ。
> ここは「計算の大陸(コンピュータ・サイエンス)」。
> 君はこれから 8 つのワールドを旅し、ダンジョンを攻略し、ボスを倒しながら、
> **大学レベルの本物のコンピュータサイエンス** を身につける。
>
> 遊び方は RPG。中身はガチの CS。誰でも始められるが、クリアした者は本物になる。

このコースは [Coding Interview University](https://github.com/jwasham/coding-interview-university) の学習内容を、
「本物の大学の CS 授業 × RPG」として再構成したものです。
各ワールドには **講義(レクチャー)**、**クエスト(演習)**、**ボス戦(実力試験)** があります。

---

## 📜 ゲームのルール

### 1. キャラクターを作る

[キャラクターシート](character-sheet.md) をコピーして、自分の冒険の記録を始めよう。
クエストをクリアするたびにチェックを入れ、XP(経験値)を加算していく。

### 2. XP とレベル

| 行動 | 獲得 XP |
|---|---|
| 📖 講義を読み切る | 10 XP |
| ⚔️ クエスト(演習)を 1 つクリア | 10〜30 XP(クエストに記載) |
| 💻 実装クエストをクリア(コードを書いて動かす) | 50 XP |
| 👹 第一大陸のボス戦に勝利 | 100 XP |
| 👹 第二大陸(学部上級)のボス戦に勝利 | 150 XP |
| 👹 天界(大学院)のボス戦に勝利 | 200 XP |
| 🏰 隠しダンジョン攻略 | 150 XP |

| レベル | 必要累計 XP | 称号 |
|---|---|---|
| Lv 1 | 0 | 見習い冒険者 |
| Lv 5 | 300 | 計算量の使い手 |
| Lv 10 | 800 | データ構造騎士 |
| Lv 20 | 1,600 | アルゴリズム魔導士 |
| Lv 30 | 2,400 | グラフの航海士 |
| Lv 40 | 3,200 | マシン語の賢者 |
| Lv 50 | 4,000 | 🏆 コンピュータサイエンス勇者(第一大陸クリア) |
| Lv 60 | 5,500 | 証明の剣士 |
| Lv 70 | 7,000 | システムの建築家 |

| Lv 90 | 10,500 | 天界の研究者 |
| Lv 99 | 13,000 | 🎓 計算機科学の大賢者(全領域制覇) |

### 3. ボス戦のルール

- ボス戦は **何も見ずに**(講義ノートを閉じて)挑戦すること
- 全問正解でなくても、**7 割以上**で勝利
- 負けたら講義に戻って再修行。**再挑戦は何度でも OK**(本物の学びは反復から生まれる)

### 4. 装備(プログラミング言語)を選ぶ

実装クエストにはコードを書く武器が必要だ。最初の武器としておすすめ:

- 🗡️ **Python** — 軽くて扱いやすい短剣。初心者に最適
- ⚔️ **Java / C++** — 重いが強力な大剣。大企業の面接でも定番
- 🏹 **JavaScript** — どこでも使える弓。ブラウザがあれば戦える

詳しくは本家の [言語リソース集](https://github.com/jwasham/coding-interview-university/blob/main/programming-language-resources.md) を参照。

## 🎓 これは「本物の授業」です

ゲームの皮をかぶっていますが、扱う内容は実際の大学 CS カリキュラムに対応しています。
**第一大陸(学部基礎)から第二大陸(学部上級の全必修)、天界(大学院の専門)まで** ——
ハーバード(CS50)・MIT・UC バークレー・スタンフォード・CMU の実際の講義に
1 対 1 で対応するよう設計してあり、独学で**学部卒〜大学院レベル**まで到達できます。

- 各講義は **定義 → 直感 → 解析 → 実装** の順で進む、大学講義と同じ構成
- クエストの演習は、実際の試験・技術面接で出題されるレベル
- 動画・教科書へのリンクは本家 [Coding Interview University](https://github.com/jwasham/coding-interview-university) の
  厳選リソース(MIT、スタンフォード、UC バークレーの公開講義など)につながっている

### 心構え(チュートリアルの長老の言葉)

> 「自分は頭が良くないから…」と思う必要はない。
> 成功しているエンジニアは賢いが、**自分は賢くないという不安を持っている者が多い**。
> 天才の物語に惑わされるな。地道に続けた者だけが魔王城にたどり着く。
> —— 詳しくは本家の [Don't feel you aren't smart enough](https://github.com/jwasham/coding-interview-university#dont-feel-you-arent-smart-enough)

### 毎日のプレイスタイル(デイリークエスト)

1. その日のテーマの講義を読む(動画があれば観る)
2. クエスト(演習)を解く
3. 実装クエストはコードを書く — **ホワイトボードか紙に書いてから打ち込むと面接力が上がる**
4. 間違えた問題は「リトライリスト」に記録し、3 日後にもう一度挑戦(間隔反復)

### 三大陸の全体像(独学で学部卒〜大学院レベルへ)

- 🏝️ **第一大陸(World 1〜8 + 魔王城)**: コーディング面接と学部基礎。まずはここを完走すれば一人前のエンジニア
- 🌄 **第二大陸(World 9〜14)**: 離散数学・OS・DB・コンパイラ・計算理論・上級アルゴリズム。大学 CS 学科の必修を網羅し、「学部卒」相当の土台が完成
- ☁️ **天界(GRAD 1〜4 + 修了式)**: 分散・セキュリティ・機械学習・アーキテクチャ。大学院・研究・産業の最前線

各大陸はその気になれば独立して挑戦できるが、伏線(通底テーマ)は順に進むほど効いてくる。
全踏破には数か月〜年単位かかる —— が、それは本物の学位と同じ重みがあるということだ。

それでは冒険を始めよう。まずは [キャラクターシート](character-sheet.md) を作成し、
[World 1: 計算量の山脈](world-01-big-o.md) へ向かえ!⚔️
